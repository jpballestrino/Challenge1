This repository contains three test cases testing the cart of amazon.com/ with different types of products. The framework used is Selenium/unittest on Python and 
it generates a HTML report to show the results.


In order run them it is required to install three packages as follows:

>> pip install HTMLTestRunner-rv
>> pip install Selenium
>> pip install webdriver-manager

Once installed this packages go to the terminal and change to the folder that contains the main.py file. Once there just run it with >>python main.py.
